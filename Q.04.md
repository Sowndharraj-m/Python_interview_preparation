Explain Python’s dynamic typing.
Python is dynamically typed, meaning:

You don’t need to declare variable types
The type is decided at runtime

x = 10
x = "Sowndhar"
In this example, the variable x is first assigned an integer value (10) and later reassigned a string value ("Sowndhar"). Python allows this flexibility because it determines the type of the variable at runtime, based on the value assigned to it. This dynamic typing feature makes Python easy to use and allows for more flexible coding styles.
