What are Python Keywords?
Python keywords are reserved words that have a specific meaning in the language and cannot be used as identifiers (such as variable names, function names, etc.). They are used to define the syntax and structure of the Python language. Examples of Python keywords include:
- `if`  
- `else`
- `for`
- `while`
- `def`
- `class`
- `import`
- `from`
- `try`
- `except`
- `raise`
- `with`
- `return`
- `yield
